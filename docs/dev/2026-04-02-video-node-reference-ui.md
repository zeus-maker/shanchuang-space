# 视频节点完成态：对齐参考图的编辑面板

## 背景与原因

首批实现仅有简易「图生视频提示词 + 按钮」，与产品参考图（分镜视频节点：悬浮工具条、文生/图生 Tab、首帧缩略、模型与规格行、积分与圆形重新生成）差距较大，需在不大改数据流的前提下补齐布局与交互骨架。

## 修改点

- `VideoNode.vue`：生成完成且带批量元数据时，节点宽度加至约 460px；预览区悬停显示「剪辑 / HD / 解析 / 下载 / 全屏」工具条（前三项暂为占位提示）；标题行配播放图标与节点名称。
- 底部面板：**文生视频** / **图生视频** 分段切换；图生模式下展示标记/运镜图标位与首帧缩略图（角标 1）；大号多行提示词；模型下拉（Pinia 全量视频模型）、比例·分辨率·时长三个下拉、语言图标触发复制提示词、「1 个」与估算 ⚡ 积分、圆形上箭头重新生成；配音开关写入 `videoGenParams.generateAudio`。
- 重新生成：图生沿用首帧/尾帧；文生仅文本与规格；提交前合并当前编辑参数到 `videoGenParams`；文生提示词落库字段 `t2vPrompt`。

## 复盘

- 剪辑 / HD / 解析 待接入真实能力时可替换为路由或弹窗。
- 首帧图为外链时若加载失败会隐藏缩略并自动切到文生 Tab，避免误点图生重试。
