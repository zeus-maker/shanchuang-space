# 图片配置节点 UI 全面改造

**日期**：2026-03-28  
**类型**：feature  
**文件**：`src/components/nodes/ImageConfigNode.vue`

## 改造目标

参考设计稿（`docs/images/8-图片节点UI.png`），将图片节点从静态表单样式改造为
"点击展开 / 点击外部收起"的动态交互面板，并新增风格广场弹框。

## 改造内容

### 1. 节点结构重新设计
- **折叠态**（默认）：仅显示标题栏 + 图片预览区 + "尝试: 图生图 / 图片高清"快捷操作
- **展开态**（点击后）：在预览区下方滑入输入面板，包含提示词输入框 + 底部控制栏
- 展开/收起由 Vue Flow 的 `selected` prop 驱动（选中 → 展开，取消选中 → 80ms 后收起）
- 过渡动画：`Transition` + CSS `translateY` + `opacity`

### 2. 风格工具栏
- 「风格」按钮：点击打开风格广场弹框，应用后在按钮上显示高亮状态 + 已选风格 Badge
- 「标记」「聚焦」：预留占位按钮
- 已应用风格的 Badge 支持一键移除（同时从提示词中删除对应后缀）

### 3. 底部控制栏（胶囊按钮组）
| 控件 | 功能 |
|------|------|
| 模型选择器 | 下拉选择图像模型，显示缩短名称 |
| 比例/画质 | 合并下拉，支持尺寸 + 画质切换（如 `1:1 · 2K`） |
| 摄像机控制 | 占位（待接入） |
| Aa | 文字增强占位 |
| 高级设置 | 占位 |
| 张数 | 1 / 2 / 4 张切换，影响 `n` 参数 |
| ⚡ 积分 | 动态计算：标准 12/张，4K 24/张，× 张数 |
| 执行 | 圆形按钮，触发 `handleGenerate('auto')` |

### 4. 风格广场弹框
- 使用 `<Teleport to="body">` 挂载到 body，避免 canvas transform 影响 `position: fixed`
- **顶部栏**：标题 + 当前模型标签 + 排序切换（最多使用/最新时间）+ 关闭
- **分类导航**：10 个分类横向滚动（推荐/摄影写真/电商营销/动漫游戏/风格插画/平面设计/建筑及室内设计/创意玩法/文创周边/小说推文）
- **Tab 切换**：全部 / 我的收藏 / 最近使用（收藏和最近使用数据持久化到 `localStorage`）
- **风格卡片网格**：2:3 比例渐变色封面 + 名称 + 使用量（万）+ Hover 显示收藏按钮
- 点击卡片 → 将风格 Prompt 后缀追加到提示词输入框，关闭弹框
- 内置 20 条 Mock 风格数据，覆盖所有分类

### 5. 多张生成支持
- 新增 `localCount` ref（默认 1，支持 1/2/4）
- `handleGenerate` 在 `n > 1` 时依次创建多个 image 节点并与配置节点连线

### 6. 图片预览
- `previewImageUrl` 计算属性：自动读取已连接输出 image 节点中的最新 URL

## 保留的原有功能
- 所有模型/画质/尺寸选择逻辑
- 连接输入（文本节点/参考图）的 prompt 拼接
- 分镜器预填 `data.prompt` 的 fallback 逻辑
- `registerCanvasGroupNodeExecuteBridge`（整组执行）
- 文件上传（触发 file input，创建 image 节点）
- 复制/删除/重命名 label
- `autoExecute` watcher
