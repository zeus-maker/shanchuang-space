# 视频节点：首帧参考图常驻展示

## 背景与原因

图生首帧 URL 已写入 `videoGenParams`，但原先仅在展开编辑且选中「图生视频」时以 56px 小图显示；折叠或切到「文生视频」时看不到参考图，不利于核对上一分镜/替换结果。

## 修改点

- 在视频预览区（加载/成片/空状态）下方增加「首帧」条：有 `first_frame_image` 即展示大图（`object-contain`，最高约 140px），折叠与文生 Tab 下同样可见；加载中也可对照首帧。
- 缩略图加载失败时显示说明文案，引导在图生内更换图片。
- 图生 Tab 内原有可点击更换的小缩略图保留。

## 复盘

展示与交互分离：大图负责「看见」，小图块负责「更换」。
